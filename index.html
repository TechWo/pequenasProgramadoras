Capstone Project - Pequeñas programadoras

En este proyecto aplicaré los conocimientos adquiridos en el curso de "Pequeñas Programadoras - Principios de Programación en JavaScript", mediante la creación de una página web estática.

Descripción del Proyecto

Este proyecto final consta de dos challenges, los cuales debes completar.

¡Dinos quién eres!


Utiliza los conocimientos adquiridos en este curso para crear una página web que hable sobre ti y que pueda decirle al mundo cómo contactarte.

¡Hagámos un "Quiz"!


Utiliza los conocimientos adquiridos en este curso para crear un Quiz con el tema de tu elección.

Recuerda que un Quiz ofrece varias respuestas para una pregunta, por lo tanto debes validar las respuestas hechas por el usuario para que al final le muestres su puntuación y calificación final.

Instrucciones

Challenge #1 - About me


¡Dile a todo el mundo quién eres y cómo pueden contactarte!

En este challenge aplicarás tus conocimientos de HTML5 y CSS, y para hacerlo sigue las instrucciones que a continuación te dejamos:

Imagina que eres una "Web Developer" desde hace mucho tiempo y quieres crear tu propia página web para que las personas puedan saber de ti, de tu trabajo, de tus gustos, aficiones e interéses y sobre todo que sepan donde pueden contactarte.

  • Abre el archivo aboutme.html en tu "browser" (chrome, firefox, safari). Da doble clic sobre el archivo, ó clic derecho y selecciona el browser de tu elección.
    ¿Qué ves?, conoce a Grace mediante su página web.
  • En tu proyecto "CapstoneProject", edita el archivo aboutme.html en el editor de textos "SublimeText". Da clic derecho en el archivo y selecciona Abrir con SublimeText.
  • Identifica los tags: < header >, < section > y < footer >. ¿Cuántos tag "section" hay?.
  • ¿Ya sabes que tags identifican cada parte de la página "About me"?.
    Edita cada tag con los datos de tu agrado, y si tienes dudas consulta a tu instructora.


Challenge #2 - Quiz


¡Haz un Quiz para tus amigos y evalúa sus conocimientos!

En este challenge aplicarás tus conocimientos de HTML5, CSS y JavaScript, para hacerlo sigue las instrucciones que a continuación te dejamos:

Imagina que eres una "Web Developer" desde hace mucho tiempo y quieres hacer una Quiz de conocimientos a tus amigos. Elige el tema de tu agrado y mános a la obra.

  • Diseñando el Quiz
    • Abre el archivo quiz.html en tu "browser" (chrome, firefox, safari). Da doble clic sobre el archivo, ó clic derecho y selecciona el browser de tu elección.
    • ¿Qué ves?, analiza la plantilla que dejamos para ti y comienza a diseñar en papel tu Quiz. Recuerda que un Quiz ofrece varias respuestas para una pregunta, por lo tanto debes validar las respuestas hechas por tus amigos para que al final les muestres su puntuación y calificación final.
    • ¿Qué más debes contemplar para tu Quiz?
      Estaría súper genial que tu Quiz supiera el "nombre" de tu amigo para darle su resultado personalizado. ¿Qué más podrías agregar?
      • Nombre del usuario
      • Correo electrónico
      • Otro
  • Editando código HTML y CSS
    • En tu proyecto "CapstoneProject", edita el archivo quiz.html en el editor de textos "SublimeText". Da clic derecho en el archivo y selecciona Abrir con SublimeText.
    • Analiza el código HTML e identifica los tags que vas a editar para crear tus preguntas y respuestas.
    • Modifica el código HTML a tu antojo. Recuerda que puedes ver los cambios que estás haciendo "actualizando" tu browser.
  • Agregando comportamiento con JavaScript
    • En tu proyecto "CapstoneProject", edita el archivo quiz.js en el editor de textos "SublimeText". Da clic derecho en el archivo y selecciona Abrir con SublimeText.
    • Analiza el código de JavaScript que ves en el archivo. ¿Qué código conoces?
    • Para agregar comportamiento, primero debes tener claro el requerimiento del problema.
      • El Quiz debe preguntar los datos al usuario y guardarlos para usarlos después.
      • El Quiz debe mostrar las preguntas y dar a elegir "varias" respuestas.
      • El Quiz debe validar que el usuario haya contestado todas las preguntas seleccionando almenos una respuesta.
      • El Quiz debe varlidar las respuestas del usuario, evaluando si fueron correctas o incorrectas.
      • El Quiz debe calcular la puntuación.
      • El Quiz debe calcular la calificación final.
      • El Quiz debe mostrar el resultado al usuario.
      • El Quiz debe proporcionar una opción para "reiniciar".

Acerca de

Aprenderé:

  • Cómo se compone una página web
  • CSS y HTML5 para crear mi contenido web
  • Usar el lenguaje de programación JavaScript para manipular mi contenido web

Aplicaré:

  • Mis conocimientos de programación en JavaScript:
    • Variables
    • Funciones
    • Ciclos
    • Estructuras de control
    • Estructuras de datos
  • Mis conocimientos de CSS:
    • rule-set
    • Selector
    • Declaration block
  • Mis conocimientos de HTML5:
    • Elements
    • Attributes
